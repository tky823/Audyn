audyn.utils.data.vctk
=====================

``audyn.utils.data.vctk`` provides utilities for VCTK dataset.

.. code-block::

    >>> from audyn.utils.data.vctk import speakers, num_speakers
    >>> print(speakers[:5])
    ['p225', 'p226', 'p227', 'p228', 'p229']
    >>> print(num_speakers)
    110
